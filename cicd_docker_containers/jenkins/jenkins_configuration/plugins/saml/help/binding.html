SAML Plugin supports two method of redirection binding HTTP-Redirect and HTTP-POST, by default HTTP-Redirect is used. Check supported binding redirection types of your IdP.
  • HTTP-Redirect - urn:oasis:names:tc:SAML:2.0:bindings:HTTP-POST
  • HTTP-POST - urn:oasis:names:tc:SAML:2.0:bindings:HTTP-Redirect
